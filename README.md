## ex07 file I/O
* 실습파일 : ex07.py
* 이름이 `func07`인 함수를 다음과 같이 완성하시오.
* 매개변수로 문자열 1개를 받아 들이시오.
* 문자열은 파일의 경로.
* 파일의 내용 예는 다음과 같음.
```
0,0.000000
1,0.017452
2,0.034899
3,0.052336
4,0.069756
5,0.087156
6,0.104528
7,0.121869
8,0.139173
9,0.156434
10,0.173648
```
* 함수의 반환값:<br>
    * `list`를 (또는 `tuple`을) 반환하시오.<br>
    * 각 항목은 `list` (또는 `tuple`) 로 파일의 각 행을 쉼표 기준으로 구분한 것.<br>
    * 문자열 대신 `float`를 담으시오.<br>
    * 반환값 예
``` python
[(0.0, 0.0),
 (1.0, 0.017452),
 (2.0, 0.034899),
 (3.0, 0.052336),
 (4.0, 0.069756),
 (5.0, 0.087156),
 (6.0, 0.104528),
 (7.0, 0.121869),
 (8.0, 0.139173),
 (9.0, 0.156434),
 (10.0, 0.173648)]
```
